Lotka-Volterra: Foxes and Hares

This is a simulation of Predator/Prey population dynamics

Try to do the following on your own

  • Make variables for hareBirth, hareDeath, foxBirth, foxDeath, hares, and foxes with values .7, .02, .02, .4, 45, and 20 respectively
  • Make a function for new Hares: hareBirth*x - hareDeath*x*y
  • Make a function for new Foxes: foxBirth*x*y - foxDeath*y
  • Use a snippet block to set up the world
  • Use a snippet block to run the simulation

The blocks you will need are under Operators and Snippets
