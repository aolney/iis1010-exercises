Learning Perceptrons

We're going to reach back into some previous classes/labs to show how we can use simple neural networks (perceptrons) to solve problems.

This is a multi-step exercise. Do the following in order:

  • Get Ex 1 block
    • Find the two blanks.
    • See if you can fill them in by looking for clues in the code and using what we talked about in class.
  • Get Ex 2 block
    • Learn the AND function.
    • Look at how many items it got right at the beginning and how many it got right as the end
    • Repeat 3 times. Notice that the number of iterations to learn changes based on the random initialization of weights (scroll down to see).
    • Repeat the above steps for OR and NOT
  • STOP FOR DISCUSSION
  • Get Ex 3 block
    • Delete the Ex 2 block by dragging it back to the palette.
    • There are two datsets for angle/turning and two for speed
    • Run each combination (turn toward/speed minimal; turn toward/speed fast; turn away/speed minimal; turn away/speed fast).
    • Notice how the behavior changes.
  • STOP FOR DISCUSSION
  • Get Ex 4 block
    • Delete the Ex 3 block.
    • Drag the Ex 2 block back ABOVE the Ex 4 block.
    • Run the Ex 4 block for AND/OR/NOT and change the Ex 2 block to match.
    • Write down the threshold for AND/OR/NOT you need to 'fix' the pseudoinverse solution.
