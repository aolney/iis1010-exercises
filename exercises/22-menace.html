Reinforcement Learning

We're going to use tic tac toe with the reinforcement learning approach in MENACE

This is a multi-step exercise. Do the following in order:

  • Get Ex 1 block. This block will embed a web page to a MENACE program.
    • Play against MENACE as a human (see drop down) for 10 games. Write down how many times you won or drew.
    • Now have MENACE play against Random about 50 times. Play it again yourself and write down how many times you won or drew.
    • Reload the page and have MENACE play against MENACE about 50 times. Play it again yourself and write down how many times you won or drew.
    • Reload the page and have MENACE play against Perfect about 50 times. Play it again yourself and write down how many times you won or drew.
  • STOP HERE FOR DISCUSSION
  • Remove the Ex 1 block
  • Get Ex 2 block. This plays a DIY version of MENACE.
    • Go down to the "Learn" function and fill in the blanks. Remember win is 3, draw is 1 and lose is -1.
    • Run the program using AutoPlay, but be sure to go slow enough that you see it win.
    • After 50 games, try playing against it yourself (clicking on the board to move). How does it compare?
    • If we have time, try changing the values for win,lose,and draw to see how performance changes.
