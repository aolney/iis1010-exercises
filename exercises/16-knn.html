KNN

We'll use the same dataset we used for creating fractals based on measurements of plants, specifically measurements of branch angle and length. The image below is a reminder of what the data labels mean.

This is a multi-step exercise. Do the following in order:

  • Get the data block
  • Get Ex 1 block
    • Run it 5 times.
    • Notice that the randomizer splits train/test differently each time.
    • Even though it is splitting them 50/50, random chance makes them unequal.
  • Get Ex 2 block and Ex 3 block
    • Run it 5 times.
    • Notice choosing a random test item with K=6 is gives neighbors of the same class.
    • Do we even need majority voting?
  • Get Ex 4 block
    • Run it 5 times.
    • Notice a synthetic example with only one feature is harder, and neighbors are different classes.
    • Now we seem to need majority voting. Even with it, we're sometimes wrong.
  • Get Ex 5 block
    • Run it 10 times and keep track of best K for easy, medium, and hard test sets
    • Is the best K different for different test sets?
    • What happens when K gets big?
  • Get Ex 6 block
    • There are 4 classes with 30 observations each, so chance is 25%
    • How well is the classifier doing on the hard test set now?
