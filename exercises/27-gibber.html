Gibber

This exercise makes music with Gibber. Gibber is a Javascript library for algorithmic music composition/performance (making music with algorithms. More information can be found at http://charlie-roberts.com/gibber/p5-gibber/

The code blocks are sequentially ordered. To make music, do the following.

  • Load p5. Click the gear icon above the editor pane and check the p5 checkbox. Now click the gear icon again and make sure Common Library is also checked.
  • Get the performance block. This block imports Gibber and creates a performance loop. To make music, enter the code that defines the music in the text entry box and hit submit. This method lets you change music without restarting the program, which is essential for live performance.
  • Each of the following blocks describes an instrument. The instrument may have effects that are defined below it. You can also kill the instrument, which is useful if you want to change a rhythm. If you kill an instrument that hasn't been played yet, you will get an error.
  • The way to make music is to drag these blocks to the editor area, switch to text editor mode, and copy (select and ctrl-c) and paste the code into the text entry box and press submit. Edit the code as needed in the editor before copy/pasting.
  • Have fun!
