Bayes Nets

We'll use Koller's "Student Example" for all exercises. The image below shows the structure of the Bayes Net and the associated conditional probability tables.

This is a multi-step exercise. Do the following in order:

  • Get the data block. This defines the Bayes net using maps/dictionaries
  • Get Ex 1 block. This calculates P(L1 | I0).
    • Run it.
    • Notice the output corresponds to each iteration of the loops AND each output represents one possible world.
    • In this case, the denominator is easy because it is not a conditional probability.
  • Ex 2. Get another Ex 1 block and modify it to calculate P(L1 | I0, DO). How does it compare to the last exercise?
  • Ex 3. Get another Ex 1 block and modify it to calculate P(D1 | GC). How does it compare to P(D1)?
  • Ex 4. Get another Ex 1 block and modify it to calculate P(I1 | GC). How does it compare to P(I1)?
  • Ex 5. Get another Ex 1 block and modify it to calculate P(I1 | GC, D1). How does it compare to the last exercise?
  • Ex 6. Get another Ex 1 block and modify it to calculate P(I1 | GB). How does it compare to P(I1)?
  • Ex 7. Get another Ex 1 block and modify it to calculate P(I1 | GB, D1). How does it compare to the last exercise?
