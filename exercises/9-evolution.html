Evolution

This is a simple model of evolution. Code samples are in Snippets 1 and 2.

Try a few things on your own to make it work:

  • Put down the evolution parameters block
  • Write a function `RandomGenome` that takes no argument and returns an array of length `genomeSize` randomly filled with 0 or 1 (see color example)
    • Get a function block, change its name, and remove its argument
    • Get the random color example, change the colors to 0,1 and change the array size to genomeSize
    • Make sure the function returns the array by making it the last line of code (you should not have an equal sign on this line)
  • Put down the `Mutate` block and type in the missing pieces to make it work
    • Mutate randomly flips a bit. If the bit is 1, it is changed to 0; if it is 0, it is changed to 1
    • Look at the loop in Mutate to decide how to fill in the missing pieces
  • Put down the main block (the last big one) and fill in population to be an array of objects of length `populationSize` with `fitness: 0` and `genome: RandomGenome()` (see people example)
    • Get the people weight/height example and change to weight/height to fitness/genome
    • Make fitness 0 to start out and genome random, e.g. `fitness: 0` and `genome: RandomGenome()`
    • Finally the make sure the array we're generating is `populationSize` long
    • Make sure your variable is called `population` and not `people`

When you run you will see the population of genomes and the total fitness in a red line.

The fitness should increase over time as the genomes are driven to 0
